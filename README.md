## 📚 What this project includes

The project demonstrates different exception handling concepts in Java:

- Basic exception handling (`try-catch`)
- Multiple exception types
- Custom exceptions
- Nested try-catch blocks
- The `finally` block
- Throwing exceptions manually
- Custom exceptions in list operations
- Try-with-resources
